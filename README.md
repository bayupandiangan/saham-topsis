📊 TOPSIS Stock Decision Support System

Sistem pendukung keputusan untuk memilih saham terbaik menggunakan metode TOPSIS (Technique for Order Preference by Similarity to Ideal Solution).
Project ini membantu pengguna dalam melakukan analisis multi-kriteria untuk menentukan saham yang paling optimal berdasarkan data yang tersedia.

🚀 Features
📈 Analisis saham berbasis multi-kriteria
⚖️ Perhitungan bobot dan normalisasi data
🎯 Penentuan solusi ideal positif & negatif
🧮 Perhitungan nilai preferensi (ranking saham)
📊 Output berupa ranking saham terbaik
🧠 Methodology

Metode yang digunakan adalah TOPSIS, dengan langkah-langkah:

Menentukan kriteria dan bobot
Membuat matriks keputusan
Normalisasi matriks
Menghitung matriks ternormalisasi terbobot
Menentukan solusi ideal positif & negatif
Menghitung jarak terhadap solusi ideal
Menghitung nilai preferensi
Melakukan perangkingan
